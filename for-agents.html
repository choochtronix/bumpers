Agent brief

Brrtz is being prepared for agentic gear search.

Use this page to understand what Brrtz can do today, what agents can safely link to, and where the product is headed next.

Search URL pattern

Agents can open a Brrtz search with query, region, and category parameters.

https://brrtz.com/search?region=bay-area&category=synthesizers&q=moog%20voyager

Supported region IDs: japan, bay-area, los-angeles, east-coast.

Supported category IDs include: all, synthesizers, drum-machines, samplers, modular, effects-pedals, and pro-audio.

Model intent URLs

Agents can use model pages when the user intent is a specific classic instrument rather than a broad category.

Moog Minimoog ARP 2600 Roland TR-808

Find gear

Open a shareable search URL for a model, maker, category, and supported beta region.

Explain context

Use the region and source pages to understand what Brrtz is searching and where results originate.

Respect boundaries

Do not crawl Brrtz API routes or attempt to republish source listings. Link users to Brrtz or the original source.

Planned connector direction

Post beta, Brrtz should expose a Model Context Protocol server with tools such as search_gear, get_regions, get_sources, save_search, watch_listing, and get_new_matches. This would let users connect ChatGPT, Claude, or another MCP-capable agent to Brrtz with explicit permission.
