# UX Loop

## Purpose

Improve usability without random redesign.

## Inputs

- Beta feedback.
- Mobile testing.
- Session notes.
- Confusing flows.
- Repeated user questions.
- Search result friction.

## Workflow

1. Identify one UX problem.
2. Define desired behavior.
3. Make the smallest useful UI change.
4. Test on mobile and desktop.
5. Confirm no regressions.
6. Update the decision log if direction changed.

## Definition Of Done

- UX problem solved or improved.
- Mobile checked.
- Desktop checked.
- No unrelated redesign.
