# Relevance Loop

## Purpose

Improve search result quality over time.

## Inputs

- Golden query tests.
- No-result searches.
- User feedback.
- Search logs if available.
- Clicked listings.
- Saved searches.
- Known synonyms and aliases.

## Workflow

1. Select a small set of search issues.
2. Add or update golden queries.
3. Modify terminology map, category intent, or ranking behavior.
4. Run tests.
5. Manually check important gear searches.
6. Document what changed.

## Definition Of Done

- Search quality improved for specific known cases.
- Tests added or updated when practical.
- No broad ranking change without validation.
- Notes added to decision log if behavior changed.
