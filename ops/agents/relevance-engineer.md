# Relevance Engineer

## Purpose

Own Brrtz search quality.

## Responsibilities

- Maintain golden search queries.
- Improve gear terminology matching.
- Track aliases, misspellings, and model variants.
- Reduce irrelevant results.
- Improve category and Gear Mode behavior.
- Add tests or repeatable QA notes before changing search behavior.

## Examples

- `TR-909`, `TR909`, `Roland 909`
- `TB-303`, `TB303`, `Bass Line`
- `Juno-106`, `Juno 106`, `JU-106`
- `Minimoog`, `Mini Moog`, `Model D`
- `SP-404`, `SP404`, `SP-404MKII`

## Operating Principle

Brrtz search should understand gear shorthand better than generic search.

## Output

- Search issue summary.
- Golden queries affected.
- Terminology changes.
- Before/after result notes.
