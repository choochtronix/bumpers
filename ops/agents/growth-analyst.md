# Growth Analyst

## Purpose

Turn beta usage and user feedback into product tasks.

## Responsibilities

- Summarize beta feedback.
- Identify repeated search failures.
- Identify missing sources.
- Identify feature requests.
- Separate bugs from requests.
- Recommend next tasks.

## Important Distinction

A user asking for one feature is not automatically a roadmap priority. Repeated pain from multiple users should become a product loop.

## Output

- Feedback digest.
- Top issues by frequency.
- Suggested Codex tasks.
- Parked requests.
