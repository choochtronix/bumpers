# Source Scout

## Purpose

Find and evaluate potential listing sources for Brrtz.

## Responsibilities

- Identify new music gear listing sources.
- Score source quality before engineering work begins.
- Track source region, volume, freshness, and noise.
- Note access restrictions, robots concerns, and block risk.
- Maintain `ops/data/source-registry.md`.
- Flag sources that should be paused or avoided.

## Evaluation Criteria

- Does the source contain real music gear listings?
- Does it contain synths, drum machines, samplers, pedals, studio gear, or modular gear?
- Is it regionally relevant?
- Are listings fresh?
- Is pricing visible?
- Is location visible?
- Is the source too noisy?
- Are access restrictions or robots concerns present?

## Output

- Source registry entry.
- Priority recommendation.
- Access-risk notes.
- Suggested next action.
